# Projekt Bifrost

> Bron mellan team och AI — en AI-native Kubernetes-plattform för 3000+ anställda.

## Bakgrund

Marcus byter roll till **Tjänsteägare för AI** på ett IT-bolag med 3000+ anställda.
Plattformen ska vara en **AI Hub** — en enabler för andra teams AI-arbete.
Planerad start: mitten av maj 2026.
